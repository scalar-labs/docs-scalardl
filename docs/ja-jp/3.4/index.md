# ScalarDL: トランザクション データベース システム用のビザンチン障害検出ミドルウェア

ScalarDL は、正確性、スケーラビリティ、およびデータベース不可知性を実現する、トランザクション データベース システム用のスケーラブルで実用的なビザンチン障害検出ミドルウェアです。
