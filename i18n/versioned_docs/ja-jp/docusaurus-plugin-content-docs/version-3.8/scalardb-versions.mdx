# ScalarDL が内部で使用する ScalarDB のバージョンおよびサポートデータベース

このドキュメントでは、ScalarDL の各バージョンが内部で使用している ScalarDB のバージョンを示します。このバージョン一覧は、以下の場合に有用です。

- ScalarDL がサポートしているデータベースについて知りたい場合。ScalarDL では、バックエンドデータベースとして [ScalarDB のサポートデータベース](https://scalardb.scalar-labs.com/docs/latest/scalardb-supported-databases/)を使用することができます。
- ScalarDL の `Function` 機能で利用可能な ScalarDB API を知りたい場合。

| ScalarDL バージョン | ScalarDB バージョン |
|:---------------|:---------------|
| 3.9.0 - 3.9.2  | 3.12           |
| 3.8.2          | 3.12           |
| 3.8.0 - 3.8.1  | 3.8            |
