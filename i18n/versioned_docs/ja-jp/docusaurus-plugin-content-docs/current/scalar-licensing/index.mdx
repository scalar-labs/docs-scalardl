# ライセンスキーの設定方法

import TranslationBanner from '/src/components/_translation-ja-jp.mdx';

<TranslationBanner />

ScalarDL Enterprise を実行するには、ライセンスキーを設定する必要があります。

- ScalarDL Enterprise エディションの商用ライセンスキーをお持ちの場合は、[商用ライセンスキーの設定方法](./commercial.mdx)を参照してライセンスキーを設定してください。
- ScalarDL Enterprise エディションを評価したい場合は、[試用ライセンスキーの設定方法](./trial.mdx)で提供されている試用ライセンスキーをご利用いただけます。
