# 商用ライセンスキーの設定方法

import TranslationBanner from '/src/components/_translation-ja-jp.mdx';

<TranslationBanner />

ScalarDL Enterprise を実行するには、`.properties` ファイルを作成し、商用ライセンスキーと証明書をそのファイルに追加する必要があります。`.properties` ファイルで、使用している製品に基づいて次の構成のいずれかをコピーし、その内容を `.properties` ファイルに貼り付けて、`<YOUR_LICENSE_KEY>` をライセンスキーに置き換えます。

:::note

- ライセンスキーをお持ちでない場合は、すぐに利用できるキーについては[試用ライセンスキーの設定方法](./trial.mdx)を、商用ライセンスの取得については[お問い合わせ](https://www.scalar-labs.com/contact)をご確認ください。
- ScalarDL のライセンス機構は、ライセンスキーの署名、有効期限、およびエディションを検証します。ただし、実行中の Pod の数や契約固有の制限などのデプロイメントの詳細は確認しません。そのため、技術的には契約で許可されている数以上の Pod をデプロイできる場合がありますが、契約で許可されている数の Pod のみを使用するようにしてください。

:::
