# ScalarDL のライブラリとツール

import TranslationBanner from '/src/components/_translation-ja-jp.mdx';

<TranslationBanner />

ScalarDL は、スケーラブルで信頼性の高いアプリケーションの構築と運用を支援するさまざまなライブラリとツールを提供します。以下は、利用可能な主要なライブラリとツールです。

## ライブラリ

ScalarDL で利用可能なライブラリは以下の通りです。

| ライブラリ                            | エディション             | Maven パッケージ                                                                                                        | コンテナイメージ  | リファレンス                                                                                                  |
|-------------------------------------|------------------------|-----------------------------------------------------------------------------------------------------------------------|-----------------|------------------------------------------------------------------------------------------------------------|
| ScalarDL Java Client SDK            | Community & Enterprise | [Maven Central Repository](https://central.sonatype.com/artifact/com.scalar-labs/scalardl-java-client-sdk)            | N/A             | [ドキュメンテーション](./how-to-write-applications.mdx#scalardl-client-sdk-を使用する)                            |
| ScalarDL HashStore Java Client SDK  | Community & Enterprise | [Maven Central Repository](https://central.sonatype.com/artifact/com.scalar-labs/scalardl-hashstore-java-client-sdk)  | N/A             | [ドキュメンテーション](./how-to-write-applications-with-hashstore.mdx#scalardl-hashstore-client-sdk-の使用)   |
| ScalarDL TableStore Java Client SDK | Community & Enterprise | [Maven Central Repository](https://central.sonatype.com/artifact/com.scalar-labs/scalardl-tablestore-java-client-sdk) | N/A             | [ドキュメンテーション](./how-to-write-applications-with-tablestore.mdx#scalardl-tablestore-client-sdk-の使用) |
