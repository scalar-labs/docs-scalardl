## v3.10.4

**発売日:** 2025年12月26日

### まとめ

このリリースには、いくつかのバグ修正および脆弱性の修正が含まれています。

### Community と Enterprise editions

#### バグの修正

- PutToMutable 関数での FLOAT および BLOB データ型の処理に関するバグを修正しました。([#297](https://github.com/scalar-labs/scalardl/pull/297))
- デジタル署名でクライアントが誤って設定された場合の NullPointerException を修正しました。([#302](https://github.com/scalar-labs/scalardl/pull/302))
- ステータスコードの処理を修正しました。([#323](https://github.com/scalar-labs/scalardl/pull/323))
- [CVE-2025-47907](https://github.com/advisories/GHSA-j5pm-7495-qmr3 "CVE-2025-47907") と [CVE-2025-58183](https://github.com/advisories/GHSA-9gcr-gp5f-jw27 "CVE-2025-58183") を修正しました。([#364](https://github.com/scalar-labs/scalardl/pull/364))
- [CVE-2025-55163](https://github.com/advisories/GHSA-prj3-ccx8-p6x4 "CVE-2025-55163") を修正しました。([#367](https://github.com/scalar-labs/scalardl/pull/367))

## v3.10.0

**発売日:** 2024年12月4日

### まとめ

このリリースでは、汎用コントラクトやファンクションなどの機能強化、およびエラーコードの追加を含むさまざまな改善が導入されています。また、いくつかのバグ修正も含まれています。詳細な変更については、以下を参照してください。

### 機能強化

- 汎用的なコントラクトとファンクションを追加しました。

### 改善点

- エラーメッセージにエラーコードを追加しました。
- 可変データベースで暗黙的な事前読み取りを有効にしました。
- 関数実行時の ScalarDB 例外処理を改善しました。
- ScalarDB バージョンを 3.14.0 にアップグレードしました。
- ノンスを使用した executeContract を非推奨にしました。

### バグの修正

- Prometheus エクスポーターの SNI ホストチェックを無効にしました。
- PKCS#8 形式の秘密鍵を処理する際のバグを修正しました。
- 予期しない検証実行を修正しました。
- ノンス以外のトランザクション ID に関するバグを修正しました。
- JDBC トランザクションマネージャーを使用したトランザクションによってアセットが誤って上書きされるバグを修正しました。
- Ledger サービスが DynamoDB でコントラクトを実行できなくなるバグを修正しました。
- ScalarDB のグループコミット機能の使用を無効にする検証を追加しました。
- Cosmos DB で実行できるように修正しました。
- 次の脆弱性を修正しました。

  - [CVE-2023-45283](https://github.com/advisories/GHSA-vvjp-q62m-2vph "CVE-2023-45283")
  - [CVE-2023-45288](https://github.com/advisories/GHSA-4v7x-pqxf-cx7m "CVE-2023-45288")
  - [CVE-2024-24790](https://github.com/advisories/GHSA-49gw-vxvf-fc2g "CVE-2024-24790")
  - [CVE-2024-34156](https://github.com/advisories/GHSA-crqm-pwhx-j97f "CVE-2024-34156")
