# 試用ライセンスキーの設定方法

import TranslationBanner from '/src/components/_translation-ja-jp.mdx';

<TranslationBanner />

ScalarDL Enterprise には以下の試用ライセンスキーをご利用いただけます。商用ライセンスキーをお持ちの場合は、[商用ライセンスキーの設定方法](./commercial.mdx)を参照してライセンスキーを設定してください。

ScalarDL Enterprise を実行するには、`.properties` ファイルを作成し、試用ライセンスキーと証明書をファイルに追加する必要があります。`.properties` ファイルには、使用している製品に基づいて以下の設定をコピーし、ファイルに貼り付けてください。

:::warning

- これらのライセンスキーは評価目的でのみ有効です。
- 試用ライセンスを使用する場合、ライセンスの検証と有効期限の確認のために、ScalarDL をインターネットに接続する必要があります。
- 試用ライセンスキーは定期的に更新されます。本番環境での使用については、[お問い合わせ](https://www.scalar-labs.com/contact)から商用ライセンスを取得してください。

:::
